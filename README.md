# Neovim config 

## For Windows

- cmd

```ps1

git clone git@github.com:iguoli/nvim-config.git %LOCALAPPDATA%/nvim
```

- Powershell

```ps1
git clone git@github.com:iguoli/nvim-config.git $env:LOCALAPPDATA/nvim
```

## For Linux

```sh
git clone git@github.com:iguoli/nvim-config.git ~/.config/nvim
```
